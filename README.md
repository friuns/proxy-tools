# Proxy Tools

A collection of Python utilities for proxy management and testing.

## Files

- `proxy_checker.py` - Check if proxies in a list are working
- `proxy_finder.py` - Fetch and test proxy lists from various sources
- `sample_proxies.txt` - Example proxy list format

## Usage

### Proxy Checker
```bash
python3 proxy_checker.py proxy_list.txt
```

### Proxy Finder
```bash
python3 proxy_finder.py
```

## Requirements

- Python 3.6+
- requests library

Install dependencies:
```bash
pip install requests
```

## Disclaimer

These tools are for educational and testing purposes. Always respect terms of service and local laws when using proxies.
